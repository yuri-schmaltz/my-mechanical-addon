# Documentacao Completa - RC Mechanism Generator

Esta pasta contem a documentacao tecnica e funcional completa do addon.

## Sumario

- `docs/visao_geral.md`
  - Objetivo, escopo, capacidades e limitacoes.
- `docs/guia_usuario.md`
  - Instalacao, configuracao, fluxo operacional e troubleshooting.
- `docs/arquitetura.md`
  - Estrutura de modulos, fluxo de dados e ciclo de vida do addon.
- `docs/referencia_operadores.md`
  - Referencia de todos os operadores Blender (`rcgen.*`).
- `docs/referencia_propriedades.md`
  - Catalogo de referencias, tolerancias e settings.
- `docs/desenvolvimento_release.md`
  - Setup de desenvolvimento, testes, qualidade e release.

## Leitura recomendada

1. Comecar por `docs/visao_geral.md`.
2. Para uso direto no Blender, seguir `docs/guia_usuario.md`.
3. Para manutencao de codigo, usar `docs/arquitetura.md` e `docs/desenvolvimento_release.md`.
4. Para automacoes e integracoes, consultar `docs/referencia_operadores.md` e `docs/referencia_propriedades.md`.
